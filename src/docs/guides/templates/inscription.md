# Checklist dossier d'inscription

Ce document liste les informations et documents à recueillir pour chaque nouvel adhérent. Le formulaire Payasso couvre la plupart de ces éléments — cette checklist sert de référence pour les cas particuliers et les vérifications.

## Informations à recueillir

### Identité

- [ ] Nom et prénom
- [ ] Date de naissance
- [ ] Adresse postale
- [ ] Email (du responsable légal pour les mineurs)
- [ ] Numéro de téléphone

### Pour les mineurs

- [ ] Nom et prénom du responsable légal
- [ ] Lien de parenté
- [ ] Téléphone d'urgence (différent du principal si possible)
- [ ] Autorisation parentale signée (dans Payasso)
- [ ] Autorisation de droit à l'image signée (dans Payasso)
- [ ] Autorisation de quitter seul les lieux (à préciser)

### Discipline(s) choisie(s)

- [ ] Karaté Shotokan (préciser tranche d'âge : 5-7 / 8-13 / Ados-Adultes)
- [ ] Karaté Défense
- [ ] Body Karaté
- [ ] Karaté Mix

### Règlement

- [ ] Cotisation réglée (via Payasso)
- [ ] Période : septembre–juin / janvier–juin / avril–juin
- [ ] Pass'Sport applicable ? (vérifier éligibilité)

### Licence FFKaraté

> La licence FFKaraté est obligatoire pour les sections Karaté Shotokan et Karaté Défense. Elle est souscrite par l'association auprès de la FFKaraté après validation de l'inscription.

- [ ] Licence à souscrire (Karaté Shotokan, Karaté Défense)
- [ ] Pas de licence requise (Body Karaté, Karaté Mix)

## RGPD

Les données collectées sont utilisées exclusivement pour la gestion des adhésions et la communication du club. Elles ne sont jamais transmises à des tiers.
